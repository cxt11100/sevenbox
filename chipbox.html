About SevenBox

SevenBox is a free online chiptune song editor. Make instrumental tracks in your browser, host a multiplayer room, and jam with friends. No account required — songs you create are yours.

NotesClick the gray rows at the top. Drag sideways to change note length.
PatternsNumbered boxes under the grid rearrange the song structure.
LoopThe loop bar sets which bars repeat. Drag the ends to cover more bars.
DrumsBottom channel row is drums. Rows above = melody & harmony.

Keyboard

Space play/pause (syncs in multiplayer) · Ctrl+F restart song for everyone · Z undo · Y redo · C/V copy/paste · 0–9 pattern · Arrows move selection

Multiplayer

Open SevenBox (same link every time). Enter a name → Host server or join a public room / code. Use Share invite to copy a link with the room code for friends. Everyone edits the same song in the browser. No install and no same Wi‑Fi required.
